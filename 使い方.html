席次表 → IDS Package 変換 使い方v2.0

席次表の PDF や画像から、会場の再生機器（Panasonic PW-DPC01）で使えるパッケージを作って、USB に直接書き出すツールです。

安心してお使いいただけます
ファイルは皆さまのパソコンの中だけで処理されます。インターネット上にアップロードされることはありません。
はじめにご準備ください
  • USB メモリ（FAT32 形式でフォーマット済みのもの。会場の機器が exFAT を認識しないため）
  • ブラウザは Google Chrome または Microsoft Edge（USB への書き込みに対応しているため）

使い方の流れ

1
ファイルを選ぶ

席次表のファイルを左側のエリアにドラッグ＆ドロップするか、「ファイルを選択」ボタンから選びます。

対応形式: PDF / JPEG / PNG

左側にファイル選択エリア、右側がプレビュー
同時に処理できるファイルは最大 2 件までです。
PDF の場合は1 ファイルあたり最大 50 ページまでです。
パソコンから席次表ファイルを選びます
2
日付・ご両家名を入れてプレビューを確認

ファイルを選ぶと、入力欄が表示されます。

  • 日付：ドロップダウンに直近 3 ヶ月の土日が並んでいます。平日や祝日のときは右のカレンダーアイコンから選んでください。
  • ご両家名：「両家のお名前を「・」でつないで」入れます（例: 神野・三本）。「家」は自動で末尾に付きます。

右側のプレビューは、会場のモニターでの見え方そのものです。左右に黒い余白が入った 16:9 の状態で表示されます。

入力するとボタンが有効になり、右側に本番イメージが出ます
左下に「ファイル名: 20260516_神野・三本家」のようにプレビューが出ます。間違いがないか確認してください。
3
USB を挿して「USB に書き出す」

パソコンに USB メモリを挿してから、青い「USB に書き出す」ボタンを押します。

フォルダ選択画面が出るので、挿した USB メモリ（例: AXE DRIVE）を選んで「選択」をクリックします。

左側のドライブ一覧から USB を選んで「選択」
注意： 必ず USB メモリを選んでください。パソコンの内蔵ディスク（Macintosh HD など）や iCloud のフォルダを選ばないでください。
4
確認ダイアログで「OK」を押す

ブラウザが「『〇〇』に書き出します。本当にこのフォルダで間違いありませんか？」と確認してきます。

USB の名前が表示されているのを確認して、「OK」を押してください。

USB の名前が間違っていないか必ず確認
USB 内に既に「IDS Package」という名前のフォルダがある場合は、上書きされます（前回書き出した内容は消えます）。
それ以外の USB の中身は触りません。
5
完了したら USB を取り外す

緑のチェックマークと「USB への書き出しが完了しました。」の表示が出たら成功です。

パソコンから USB を取り外し、会場の再生機器（PW-DPC01）に挿してご使用ください。

この画面が出たら完了です
続けて別の席次表を変換したい場合は、「続けて別のファイルを変換」ボタンを押すと最初の画面に戻ります。

よくある質問

USB を選ぶ画面で USB が出てこない

USB のフォーマットが FAT32（MS-DOS）になっているか確認してください。exFAT や NTFS だと会場の機器（PW-DPC01）が認識できません。フォーマットし直すと中のデータは全部消えるので注意してください。

「このブラウザは USB への直接書き出しに対応していません」と出る

Safari や Firefox では USB に直接書き出せません。Google Chrome か Microsoft Edge でお試しください。

変換に時間がかかります

大きな PDF やページ数の多い PDF は時間がかかります。途中でタブやブラウザを閉じないでください。1 ファイル 50 ページが上限です。

会場のモニターで席次表の左右が白くなってしまう

v2.0 では左右が黒の余白になるように修正されています。もし白い場合は、ブラウザを Cmd + Shift + R（Windows は Ctrl + F5）で強制再読み込みして、画面上部に v2.0 のバッジが表示されているか確認してください。

うまく動かないとき

ページを Cmd + Shift + R（Windows は Ctrl + F5）で再読み込みしてみてください。それでも改善しない場合は管理者までご連絡ください。
